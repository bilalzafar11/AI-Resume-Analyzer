# AI Resume Analyzer

## Overview

AI Resume Analyzer is a web application that helps users check how well their resume matches a specific job description. It analyzes resumes, detects important skills, calculates an ATS-style match score, and provides suggestions to improve the resume before applying for jobs.

I built this project to practice Python, FastAPI, and AI concepts while creating a practical tool that can help job seekers improve their resumes.

## Features

* Upload resumes in PDF or DOCX format
* Extract text from uploaded resumes
* Detect technical skills automatically
* Compare resume skills with a job description
* Calculate an ATS-style match score
* Display matched and missing skills
* Generate AI-based suggestions for resume improvement
* Clean and responsive user interface

## Tech Stack

* Python
* FastAPI
* HTML
* CSS
* JavaScript
* Jinja2
* PDF & DOCX Text Extraction
* AI-Based Skill Detection

## How It Works

1. Upload your resume in PDF or DOCX format.
2. Enter the job description.
3. The application extracts text from the resume.
4. It detects the skills mentioned in the resume.
5. The detected skills are compared with the required job skills.
6. An ATS match score, missing skills, and improvement suggestions are displayed.
